入门
-------------------------

在浏览器打开[reader/index.html](http://futurepress.github.com/epubjs-reader/index.html) 

您可以通过在 URL 中输入指向 bookPath 的链接来更改它打开的 ePub：

`?bookPath=https://s3.amazonaws.com/epubjs/books/alice.epub`
